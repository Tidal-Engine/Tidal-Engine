Quick Start Guide

Get up and running with the Tidal Engine quickly. This guide covers installation, building, and basic development setup.

Prerequisites

System Requirements

ComponentMinimumRecommended
OSLinux, Windows 10, macOS 10.15Latest versions
CPU4-core 64-bit processor8-core modern CPU
RAM8GB16GB+
GPUVulkan 1.3 compatibleDedicated GPU with 4GB+ VRAM
Storage2GB free spaceSSD with 10GB+ free

Development Tools

  • C++23 Compiler: GCC 13+, Clang 16+, or MSVC 19.35+
  • CMake: Version 3.25 or higher
  • Vulkan SDK: Latest version from LunarG
  • Git: For source control

Installation

Linux (Ubuntu/Debian)

# Update package list
sudo apt update

# Install build essentials
sudo apt install build-essential cmake ninja-build git

# Install Vulkan SDK
sudo apt install vulkan-sdk vulkan-validationlayers-dev

# Install dependencies
sudo apt install libglfw3-dev libglm-dev libstb-dev

# Verify Vulkan installation
vulkaninfo

Linux (Fedora)

# Install development tools
sudo dnf install gcc-c++ cmake ninja-build git

# Install Vulkan SDK
sudo dnf install vulkan-devel vulkan-validation-layers-devel

# Install dependencies
sudo dnf install glfw-devel glm-devel stb_image-devel

# Verify installation
vulkaninfo

Windows

Recommended: Use Visual Studio 2022 with vcpkg for dependency management. 
# Install Visual Studio 2022 with C++ workload
# Download and install Vulkan SDK from LunarG

# Install vcpkg
git clone https://github.com/Microsoft/vcpkg.git
cd vcpkg
.\bootstrap-vcpkg.bat

# Install dependencies via vcpkg
.\vcpkg install glfw3:x64-windows
.\vcpkg install glm:x64-windows
.\vcpkg install stb:x64-windows

macOS

# Install Xcode command line tools
xcode-select --install

# Install Homebrew (if not already installed)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install dependencies
brew install cmake ninja git
brew install molten-vk  # Vulkan on macOS
brew install glfw glm

# Install Vulkan SDK from LunarG website

Building the Engine

Clone the Repository

# Clone the source code
git clone https://github.com/your-username/Tidal-Engine.git
cd Tidal-Engine

# Check available branches
git branch -a

Configure with CMake

System Dependencies (Linux/macOS)

# Create build directory
mkdir build && cd build

# Configure with system packages
cmake .. -GNinja -DCMAKE_BUILD_TYPE=Debug

# Or configure for release
cmake .. -GNinja -DCMAKE_BUILD_TYPE=Release

vcpkg Dependencies (Windows/Cross-platform)

# Configure with vcpkg toolchain
cmake .. -GNinja ^
    -DCMAKE_BUILD_TYPE=Debug ^
    -DCMAKE_TOOLCHAIN_FILE=C:\path\to\vcpkg\scripts\buildsystems\vcpkg.cmake

Build the Project

# Build with Ninja (recommended)
ninja

# Or build with make (if using make generator)
make -j$(nproc)

# Or build with Visual Studio
cmake --build . --config Debug

Run the Engine

# Run the main application
./TidalEngine

# Run dedicated server
./TidalEngine --server

# Connect to remote server
./TidalEngine --client 192.168.1.100

Development Setup

Debug Configuration

Important: Always use debug builds during development to enable Vulkan validation layers. 
# Debug build with validation layers
cmake .. -DCMAKE_BUILD_TYPE=Debug -DENABLE_VALIDATION_LAYERS=ON

# Enable additional debugging
cmake .. -DCMAKE_BUILD_TYPE=Debug \
    -DENABLE_VALIDATION_LAYERS=ON \
    -DENABLE_DEBUG_MARKERS=ON

Common Issues

Build Problems

"Vulkan not found":
• Ensure Vulkan SDK is installed and in PATH
• Check that VK_SDK_PATH environment variable is set
• Verify vulkaninfo command works
"GLFW not found":
• Install GLFW development packages
• Use vcpkg for Windows: vcpkg install glfw3
• Check CMake can find the library

Runtime Issues

"Validation layer error":
• Normal in debug builds - shows API usage issues
• Fix the underlying Vulkan API misuse
• Disable with -DENABLE_VALIDATION_LAYERS=OFF for testing
"Window creation failed":
• Update graphics drivers
• Check display server (X11/Wayland on Linux)
• Verify Vulkan support: vulkaninfo
